Για να κατεβασεις το project αρχικα:
1 κατεβαζεις ολα τα αρχεια που εχει στο github και τα βαζεις σε ενα φακελο 
2 ανοιγεις το Unity hub και πατας add > add project from disk
3 βρισκεις τον φακελο που κατεβασες και τον ανοιγεις 
4 μολις ανοιξει πρεπει να κατεβασεις ολα τα assets που εχουμε χρησιμοποιησει βαζω Links εδω
https://assetstore.unity.com/packages/3d/environments/fantasy-landscape-103573
https://assetstore.unity.com/packages/3d/environments/medieval-castle-227378
https://assetstore.unity.com/packages/3d/characters/humanoids/fantasy/free-low-poly-human-rpg-character-219979
https://assetstore.unity.com/packages/3d/environments/stylize-modular-house-demo-free-erbeilo-3d-220218

5 οταν κατεβουν ολα πρεπει να πας στο file > open Scene και να βρεις το αρχειο OutdoorsScene.unity που ειναι στο φακελο assets μεσα στα αρχεια που κατεβασες.
6 μολις ανοιξεις το αρχειο αυτο θα εμφανιστουν πραγματα που λογικα θα ειναι ολα ροζ
7 για να τα φτιαξεις πρεπει να πας στο edit > rendering > materials > convert all built-in materials to HDRP
λογικα μετα απο αυτο θα λειτουργει κανονικα
